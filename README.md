- 🔥 **Java 核心** - 集合、并发、JVM、IO/NIO
- 🌱 **Spring 生态** - Spring、Spring Boot、Spring Cloud
- 🏗️ **微服务** - 架构设计、服务治理、分布式事务
- 💾 **数据库** - MySQL、Redis、SQL 调优
- ⚡ **性能优化** - JVM 调优、缓存优化、接口优化
